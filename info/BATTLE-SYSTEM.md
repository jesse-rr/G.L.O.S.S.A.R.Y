### “Living Language” Combat System

- You construct words of power from ancient symbols.
- Each rune/image/symbol = a syllable
- Combination generates chains (attacks).
- Longer chains = stronger effects.

### Example Gameplay Loop:

1. Each turn you get runes like:
```bash
ᚠ = energy
ᚱ = motion
ᚲ = destruction
ᛞ = defense
```
2. Player builds a sequence:
```bash
ᚠ + ᚲ = Fire blast
ᚱ + ᚲ = Dash attack
ᛞ + ᚠ = Barrier recharge
```
